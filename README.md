# Automatic_control_theory
Анализ наблюдаемости и управляемости системы дифференциальных уравнений в области теории автоматического управления.

Репозиторий: [Vladblast/Automatic_control_theory](https://github.com/Vladblast/Automatic_control_theory)

## О проекте
В этом репозитории собраны Jupyter-ноутбуки (Python) для практического анализа свойств линейных систем управления:
- проверка управляемости и наблюдаемости;
- вычисление рангов матриц Калмана;
- применение критериев Калмана и Попова–Хаутуса;
- демонстрация на численных примерах.

Цель — дать компактные и воспроизводимые примеры вычислений, которые можно быстро адаптировать под свою систему.

## Теоретические основы (кратко)
Рассматривается линейная стационарная система в пространстве состояний:
- непрерывная: 
  
  $\dot{x} = Ax + Bu,\quad y = Cx + Du$
  
-  дискретная:
  
  $\dot{x_{k+1}} = Ax_k + Bu_k,\quad y_k = Cx_k + Du_k$
  

- Управляемость: система полностью управляемая, если ранг матрицы управляемости
  
  $\mathcal{C} = [\,B\ \ AB\ \ A^2B\ \ \ldots\ \ A^{n-1}B\,]$
  
  равен $n$ (размерности состояния).

- Наблюдаемость: система полностью наблюдаемая, если ранг матрицы наблюдаемости

$$
\mathcal{O} = \begin{bmatrix}
C \\
CA \\
CA^2 \\
\vdots \\
CA^{n-1}
\end{bmatrix}
$$

  равен $n$.

- Критерий Попова–Хаутуса:
  - управляемость ⇔ для каждого собственного значения $\lambda$ матрицы $A$ выполняется
$\mathrm{rank}[\lambda I - A \mid B] = n$

  - наблюдаемость ⇔ для каждого $\lambda$ выполняется
    $\mathrm{rank}\[\ \lambda I - A\ \mid\ C^\top\ ] = n$ (или эквивалентно по транспонированию).

В ноутбуках демонстрируются вычисления этих критериев и интерпретация результатов.

## Как запустить
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Vladblast/Automatic_control_theory.git
   cd Automatic_control_theory
   ```
2. Установите зависимости (см. раздел «Требования»).
3. Запустите Jupyter:
   ```bash
   jupyter notebook
   ```
4. Откройте интересующий `.ipynb` и последовательно выполните ячейки.

## Как использовать ноутбуки
- В первых ячейках обычно задаются матрицы системы `A`, `B`, `C`, `D`.
- Измените эти матрицы под вашу задачу и запустите последующие ячейки для проверки управляемости/наблюдаемости.
- Результаты сопровождаются промежуточными вычислениями (ранги, собственные значения, матрицы Калмана и т.д.), что упрощает интерпретацию.
